# escalas

mensagens.js
O arquivo mensagens.js que contem funções genéricas para lidar com mensagens, sem se preocupar com o estilo.
Há arquivo CSS separado que defina o estilo das mensagens para cada funcionalidade.
Permitir que as funcionalidades específicas definam seu próprio estilo para as mensagens, caso necessário.
Em resumo, o arquivo mensagens.js principal que lida com todas as mensagens do sistema, e que pode ser usado por diferentes funcionalidades. O CSS pode ser tratado de forma separada, dependendo das necessidades específicas de cada funcionalidade.
